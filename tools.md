## ☕ Java

### Verwendung:
- Backend-Logik
- API-Kommunikation (TRIAS)
- Datenverarbeitung
- Zustandsberechnung der Züge

### Was du konkret nutzt:
- Klassen & Objekte
- Listen (z. B. ArrayList)
- Zeitberechnung (LocalDateTime)
- HTTP Requests
- JSON erstellen

### Was du lernen solltest:
- Grundlagen (Klassen, Methoden, Schleifen)
- OOP (Objektorientierung)
- Collections (Listen, Maps)
- REST APIs
- JSON

### Wie es funktioniert:
Java-Code wird kompiliert und läuft auf der JVM (Java Virtual Machine).  
Dadurch ist er plattformunabhängig.

### Kosten / Nutzung:
- ✅ Kostenlos (OpenJDK)
- ✅ Kommerzielle Nutzung erlaubt
